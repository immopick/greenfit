# GreenFit — Site vitrine salle de fitness

Site vitrine moderne pour une salle de fitness, développé en **React + Vite**.
Palette : blanc / violet / vert. Design responsive et animations au scroll.

## Sections

- **Hero** avec vidéo en arrière-plan
- **Le centre** (présentation)
- **Cours collectifs**
- **Planning de la semaine** (filtrable par jour)
- **Tarifs** (mensuel / annuel)
- **FAQ** (accordéon)
- **Contact** (formulaire)

## Démarrer

```bash
npm install
npm run dev
```

Le site est dispo sur `http://localhost:5173`.

## Vidéo du Hero

Dépose ta vidéo dans `public/videos/hero.mp4`.
En l'absence de vidéo, une image de secours s'affiche automatiquement.

## Build de production

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 5
- CSS natif (variables, grid, flexbox) — aucune dépendance UI
